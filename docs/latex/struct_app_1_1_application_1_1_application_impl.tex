Run the application. 


\begin{DoxyParams}{Parameters}
{\em demo} & True if the application is running in demo mode, false otherwise.\\
\hline
\end{DoxyParams}
Run the application loop using the following steps\+:
\begin{DoxyEnumerate}
\item Poll GLFW events for input processing.
\item Check if the window is minimized and prevent rendering if it is.
\item Create a new Im\+Gui frame.
\item Render either the demo or the actual application window.
\item Check for Im\+Gui state changes.
\item Render the texture for the Im\+Gui render window.
\item Get Im\+Gui background color and render the scene.
\item Draw Im\+Gui to Open\+GL window and swap buffers to present frame to screen. 
\end{DoxyEnumerate}
