void App::ImGuiManager::ImGuiManagerImpl::CreateBaseWindow ( )

Create the base window.

Create the base window of the ImGuiManager. Add the NoDocking flag to the window flags to prevent docking of the main window. Then get the main viewport, assign the position and size of the window to the viewport, and set the viewport. Finally create the Application specific dockspace.

Creates the ImGui dockspace to allow window docking. Defines the structure of the dockspace for each window and locks the windows in place, but allows for resizing.

◆ CreateSelectionWindow()

void App::ImGuiManager::ImGuiManagerImpl::CreateSelectionWindow ( )

Create the simulation selection window.

Create the simulation selection window. Creates drop down menus that allow the selection of the simulation type and subsimulation type. Then creates the simulation specific parameters to define the simulation. Finally, adds two buttons to setup and start the simulation.
